# chat
Если вы устанавливаете этот "Проект" на android устройство, то вам нужно:
1. Установить termux api, termux widget, termux
2. Выполните команду
curl https://raw.githubusercontent.com/itoryon/meow/main/install -o install && curl https://raw.githubusercontent.com/itoryon/meow/main/install2 -o install2 && chmod +x install && chmod +x install2 && ./install2 && ./install
   
8. После можете сделать termux widget (если не появился chat то перезагрузите виджет.
9. Запускаете из виджета или из ~ директории командой ./meoww
