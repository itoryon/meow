# chat
Если вы устанавливаете этот "Проект" на android устройство, то вам нужно:
1. Установить termux api, termux widget, termux
2. Выполните команду
curl https://raw.githubusercontent.com/itoryon/meow/main/install -o install && curl https://raw.githubusercontent.com/itoryon/meow/main/install -o install2.sh && chmod +x install && chmod +x install2.sh
3. Выполните команду
mv storage/downloads/install.txt ~/install && mv storage/downloads/install2.sh.txt ~/install2.sh && cd && chmod +x install && chmod +x install2.sh ./install2.ch && ./install
   
8. После можете сделать termux widget (если не появился chat то перезагрузите виджет.
9. Запускаете из виджета или из ~ директории командой ./meoww
